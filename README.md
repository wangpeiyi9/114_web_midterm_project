# 114_web_midterm_project
## 專案名稱
餐廳線上預約系統
## 組員
王培怡、李千軍
## 專案簡介
此專案為餐廳線上預約系統範例，使用者可以於網頁中填寫訂位資訊（姓名、電話、Email、用餐目的、人數、日期、時段、備註），系統會進行表單驗證，並在送出前顯示確認視窗。確認後資料會儲存在 `localStorage`。若某時段已達 3 組預約，該時段會顯示為「已滿」並不可選。
## 使用技術
- HTML5（語意化標籤）
- CSS3 / Bootstrap 5（響應式設計）
- JavaScript（即時互動、DOM 操作）
- Constraint Validation API
- localStorage（儲存預約資料）
## 功能特色
1. 表單即時與自訂驗證（電話/Email）
2. 動態產生「今天起一週內」日期
3. 動態產生半小時時段（10:00 ~ 20:00）
4. 時段滿位（≥3 組）反灰不可選
5. 送出前顯示確認視窗（modal），確認後儲存在 localStorage
6. 深色模式切換、動態動畫、完整 RWD
